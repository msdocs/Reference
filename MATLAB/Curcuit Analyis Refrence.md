##Basics

###Simple Arithmetic

```
EDU>> 4+4

```
###Setting Variables 

```
EDU>> x=4

x =
     4

EDU>> y=7

y =
     7

EDU>> x * y

ans =
    28
```

###Creating a Matrix
````
x= [5 3 10;
5 7 4;
0 0 0]
x =
     5     3    10
     5     7     4
     0     0     0
````
###Creating an Algebraic Function 

```
x = 7
y= 2*x+7
x =

     7
y =

    21
```
